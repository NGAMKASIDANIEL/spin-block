# Student Quiz Portal

A simple web app with:

- Signup and login
- University, course, and level profile setup
- Course selection
- Quiz questions
- Score and failed-question review

## Run locally

1. Open a terminal in this folder
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`

## Deploy online

You can deploy this app to a Node.js host like Render, Railway, or Vercel. Just push the repository to GitHub and connect the service.
